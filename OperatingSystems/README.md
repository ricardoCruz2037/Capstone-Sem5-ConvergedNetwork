# Sistemas Operativos — Operating Systems

Este módulo explora la **gestión de procesos e hilos** en entornos multi-OS, con el propósito de optimizar el rendimiento del sistema en los nodos del proyecto.

## Objetivos
- Comprender la interacción entre procesos distribuidos.  
- Implementar **concurrencia y sincronización de hilos**.  
- Evaluar el impacto de la gestión de recursos en la red.  

## Contenido
| Elemento | Descripción |
|-----------|-------------|
| `pruebas/` | Scripts y capturas de gestión de procesos. |
| `configs/` | Configuraciones del sistema (Linux/Windows). |
| `reporte-so.md` | Documento técnico de análisis y resultados. |

## Tecnologías Utilizadas
- Python (threading, multiprocessing)  
- Linux / Windows Server  
- Herramientas de monitoreo de procesos  

## Competencias Aplicadas
- Sincronización y control de procesos.  
- Gestión eficiente de CPU y memoria.  
- Comparativa de rendimiento multi-OS.

---
> *La eficiencia en la gestión de procesos es clave para mantener estable el sistema distribuido del proyecto.*
