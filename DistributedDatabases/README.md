# Bases de Datos Distribuidas — Diseño e Implementación

Este módulo aborda el diseño lógico, fragmentación e implementación de **bases de datos distribuidas** alojadas en entornos multi-OS.

## Objetivos
- Diseñar una base de datos distribuida con **fragmentación horizontal y vertical**.
- Implementar **replicación** para garantizar disponibilidad de datos.
- Asegurar la comunicación entre nodos mediante IP fija y segmentación VLAN.
- Integrar la base de datos con la aplicación web del módulo de desarrollo.

## Contenido
| Elemento | Descripción |
|-----------|-------------|
| `scripts/` | Scripts SQL para creación, fragmentación y replicación. |
| `docs/` | Diagramas entidad-relación y esquemas lógicos. |
| `reporte-bd.md` | Reporte técnico de diseño, normalización y configuración. |

## Tecnologías Utilizadas
- Microsoft SQL Server   
- Herramientas de replicación y fragmentación  

## Competencias Aplicadas
- Diseño e implementación de bases de datos distribuidas.  
- Optimización mediante índices y consultas remotas.  
- Sincronización entre servidores.  
- Seguridad en conexiones SQL.

> *El diseño distribuido garantiza disponibilidad y eficiencia en los procesos de la red convergente.*
