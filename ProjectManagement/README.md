# Gestión de Proyectos — Planning and Documentation

Este módulo reúne los elementos de **planificación, organización y control** del proyecto Capstone.

## Objetivos
- Planificar el desarrollo mediante un **diagrama de Gantt**.  
- Estimar recursos humanos, materiales y financieros.  
- Asignar roles y responsabilidades dentro del equipo.  

## Contenido
| Elemento | Descripción |
|-----------|-------------|
| `gantt.png` | Diagrama de Gantt del proyecto. |
| `presupuesto.xlsx` | Planificación financiera y de recursos. |
| `roles-equipo.md` | Asignación de roles y funciones. |

## Competencias Aplicadas
- Gestión del tiempo y recursos.  
- Coordinación interdisciplinaria.  
- Evaluación de riesgos y control de entregables.

---
> *El módulo de gestión asegura el cumplimiento de objetivos, tiempos y estándares del proyecto integrador.*
