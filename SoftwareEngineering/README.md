# Ingenieria en software — Software Engineering

Este módulo comprende el desarrollo del **sistema web** que interactúa con las bases de datos distribuidas a través de la red convergente implementada.

## Objetivos
- Crear una **interfaz web multiplataforma** para gestión de datos.
- Conectar la aplicación con las bases de datos distribuidas.  
- Integrar mecanismos de autenticación y control de acceso.  
- Implementar un diseño sustentable y accesible.

## Contenido
| Elemento | Descripción |
|-----------|-------------|
| `app/` | Código fuente de la aplicación (backend y frontend). |
| `templates/` | Archivos HTML/Jinja para las vistas. |
| `static/` | Recursos estáticos (CSS, JS, imágenes). |
| `reporte-desarrollo.md` | Documento técnico del sistema web. |

## Tecnologías Utilizadas
- Python (Flask / Django)  
- HTML, CSS, JavaScript  
- SQLAlchemy / PyODBC  
- Bootstrap / Tailwind (UI design)  

## Competencias Aplicadas
- Integración entre backend y bases de datos.  
- Diseño responsivo y accesible.  
- Seguridad en sesiones y formularios.  
- Pruebas de conectividad con la red convergente.

---
> *La aplicación sirve como capa de interacción entre el usuario final y los servicios distribuidos de la red institucional.*
